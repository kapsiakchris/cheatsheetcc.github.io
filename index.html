CSS Display Property

The display property specifies the type of rendering box of an element

Value Definition example
none The element is completely removed {display: none;}
inline Display element inline, height and width properties have no effect {display: inline;}
block Displays element as block element, similar to <p> {display: block;}
inline-block Displays as inline block element that you can apply height and width values to {display: inline-block;}
